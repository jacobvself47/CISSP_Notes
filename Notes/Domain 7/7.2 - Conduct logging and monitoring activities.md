## User and Entity Behavior Analytics (UEBA)
- UEBA - User and Entity Behavior Analytics
	- entity (user or system) behavior is collected and input into a threat model
	- the model establishes a baseline of normal based on historical data
	- enables analysis to uncover more details about anomalous events
	- Usually includes automated investigation
